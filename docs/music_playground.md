# Music Playground 音乐生成器

Music Playground 会将训练好的音乐 checkpoint 转换为一套和弦生成工作流。它位于和弦预处理流水线之上，作为面向用户的产品交互层。

## 入口

打开 Web UI，并在侧边栏中选择 **音乐生成**：

```text
http://127.0.0.1:3001/music
```

## 使用流程

1. 选择一个 checkpoint。如果存在音乐领域 checkpoint，系统会优先推荐。
2. 点击 **加载模型**，将选中的 checkpoint 加载到聊天后端。
3. 选择一个预设，例如 `流行循环`、`华语抒情`、`爵士转接`、`Neo-soul` 或 `电影感小调`。
4. 调整 `风格`、`情绪`、`调性`、`起始和弦 / 罗马数字` 和 `BPM`。
5. 点击 **生成和弦**。
6. 查看生成的和弦时间线。

## 输出解析

页面会要求模型按以下格式返回单行结果：

```text
Roman: <roman numerals with | bar separators> | Progression: <playable chords>
```

时间线解析器会优先读取 `Roman:` 部分。如果该部分缺失，则回退读取 `Progression:`；如果仍无法解析，则使用模型生成的原始文本。

## 试听播放

页面内置了轻量级 Web Audio 试听功能：

- 点击时间线卡片，可以试听单个和弦。
- 点击 **试听整段**，可以按当前 BPM 播放完整和弦进行。
- 罗马数字会根据当前选择的调性解析为实际和弦。
- 也支持 `C`、`Am`、`F#dim`、`Gaug` 等绝对和弦写法。

## MIDI 导出

点击 **导出 MIDI**，可以下载一个简单的 type-0 MIDI 文件：

- 时间线中的每个和弦都会写入为三和弦。
- 每个和弦持续两拍。
- MIDI 速度会跟随当前选择的 BPM。

## 分享配置

点击 **分享配置**，可以复制一个可复现当前配置的 URL，内容包括：

- 已选择的预设
- 风格
- 情绪
- 调性
- 起始和弦进行
- BPM
- checkpoint 描述信息

生成的音频或 MIDI 结果不会嵌入 URL。打开分享链接后，需要重新生成。
